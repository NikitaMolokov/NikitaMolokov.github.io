Joistick

JoyStick является аналоговым датчиком, который может быть использован
для управления вашим arduino.
Аналоговые JoySticks в основном potentiometers,поэтому они возвращают значения,
которые могут быть сделаны видимыми с помощью предоставленного кода на этом этапе.
Нажав на серийный монитор, вы увидите значения. Как только вы переместите JoyStick значения изменятся.
Этот код также удобен, если вы хотите управлять arduino с помощью JoyStick.
